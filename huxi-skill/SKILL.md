# 简讯写作技能

本技能用于根据提供的材料、信息和写作要求，撰写符合规范的机关单位简讯。

## 核心文件

- **jianxun.md** - 简讯写作规范知识库，包含行文逻辑、标题格式、语言组织等所有规范
- **references/** - 参考范文目录，每次写作前应熟读学习

## 两个核心指令

### 指令一：学习

当用户说"学习"、"更新规范"、"总结写作方式"时执行：

1. 读取 `jianxun.md` 了解当前规范
2. 读取 `references/` 目录下的新范文的
3. 分析新范文的：
   - 标题结构与命名方式
   - 导语写作模式（五要素）
   - 主体部分的叙述逻辑
   - 序号合作的表述方式
   - 结语的固定套路
   - 常用词汇和搭配
4. 将学到的新的写作方式更新到 `jianxun.md` 中
5. 告知用户学到了什么新特点

### 指令二：写作

当用户说"写简讯"、"起草简讯"、"帮我写"时执行：

**Step 1：准备工作**
1. 先阅读 `jianxun.md` 规范
2. 阅读 2-3 篇参考范文（优先选择与任务最相关的类型）

**Step 2：确认信息**
向用户确认以下必要信息（缺少任何一项都要主动询问）：

| 必要信息 | 说明 |
|---------|------|
| 时间 | 活动发生的具体日期和时间 |
| 主要领导 | 带队领导的姓名和职务 |
| 地点/单位 | 访问/考察的地点或单位 |
| 活动名称 | 具体活动类型 |
| 活动目的 | 简短的背景/目的说明 |
| 参与方 | 其他参加的领导或单位 |
| 活动详情 | 座谈内容、企业介绍、领导讲话要点等 |

**Step 3：撰写简讯**
按照 `jianxun.md` 中的规范结构撰写：

```
【主标题】（8-12字，对仗式或概括式）
副标题：XXX赴XXX开展XXX活动

导语（100字内）：时间+人物+地点+活动+目的

主体部分：
- 被访方致欢迎辞+介绍情况
- 我方分享经验/介绍工作
- 领导讲话要点（精炼保留核心）

具体工作方向（序号列表，3-5条为宜）：
一是......
二是......
三是......

结语：肯定成效+明晰方向+后续行动+最终目标
```

**Step 4：提交审查**
- 完成初稿后提交给用户
- 根据用户反馈修改
- 注意：如果用户提到"领导说不行"、"需要调整"等，要主动询问具体问题和修改方向

## 注意事项

1. **信息不全时必须询问**：不要凭空编造信息，简讯讲究真实准确
2. **语言要正式规范**：符合政府机关公文风格，避免口语化
3. **领导讲话要精炼**：保留核心观点，不要大段堆砌
4. **篇幅要适中**：一般500-1500字，根据活动重要性调整
5. **序号不宜过多**：合作方向3-5条为宜，突出重点
