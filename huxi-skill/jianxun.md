# 简讯写作规范

> 本文件是简讯写作的知识库，由"学习"指令维护更新。记录简讯的写作规律、行文逻辑和语言组织方式。

---

## 一、简讯定义与定位

简讯是机关、单位用于宣传报道的简短新闻体裁，具有以下特点：
- **时效性强**：快速反映工作动态
- **篇幅简短**：一般500-1500字
- **语言精炼**：用最少的字表达最准确的信息
- **格式规范**：有相对固定的写作模式

---

## 二、标题格式

### 标准结构

简讯标题通常由**主标题 + 副标题**组成，形成"主题句 + 活动说明"的双层结构：

```
【主标题】（8-12字，概括核心主旨）
副标题：XXX赴XXX开展XXX活动
```

### 主标题命名规律

#### 1. 对仗式（最常用）

四字对八字，讲究平仄对称：

| 主标题 | 说明 |
|--------|------|
| 深化协同创新 共筑发展高地 | 六字对六字 |
| 以赛引赛融资源 赛事协同筑高地 | 七字对七字 |
| 深化赛事成果 激发创新活力 | 六字对六字 |
| 交流互鉴 共谋发展 | 四字对四字 |
| 深化协同创新 共筑发展高地 | 六字对六字 |

#### 2. 概括式（单句）

直接概括活动或人物：

- "赵延忠赴下陆区开展调研考察"
- "闫树调研武汉地质资源环境工业技术研究院氢能项目"
- "理工对接简讯"

#### 3. 新闻式（带符号，更活泼）

- "赛后服务│赴成都开展大健康产业招商考察"
- "跨域'揽智'火力全开！80余个优质项目争先加入"

#### 4. 会议/筹备类标题

- "第二届黄石未来产业创新创业大赛工作推进会顺利召开"
- "黄石未来产业创新创业大赛组委会召开大赛筹备会"

### 副标题标准写法

```
[领导/单位名称]赴[地点/单位]开展[活动名称]
```

示例：
- "市创新发展中心赴东楚投资集团开展低空经济学习交流活动"
- "黄石市创新发展中心赴湖北理工学院开展对接交流"

## 三、行文结构

### 标准五段式

```
┌─────────────────────────────────────────┐
│ 第一段：标题（含主标题+副标题）           │
├─────────────────────────────────────────┤
│ 第二段：导语（时间+人物+地点+活动+目的）   │
├─────────────────────────────────────────┤
│ 第三段：主体（活动详情+领导讲话+座谈内容） │
├─────────────────────────────────────────┤
│ 第四段：具体工作/合作方向（常用序号列表） │
├─────────────────────────────────────────┤
│ 第五段：结语（总结+展望）                 │
└─────────────────────────────────────────┘
```

### 各段写作要点

#### 第二段：导语

**必备要素**（五W）：
- **时间**：X月X日上午/下午、X月X日至X日
- **人物**：主要领导姓名+职务
- **地点**：访问/考察地点
- **活动**：具体活动名称
- **目的**："为......"

**句式模板**：
```
为[目的]，[时间**或**具体日期**或**日期范围]，[主要领导姓名+职务]带队/率队赴[地点]开展[活动]。[相关领导/单位]参加活动。
```

**示例**：

示例1（标准）：
```
为深入推进校地合作，精准对接高校创新资源服务地方产业发展，5月8日，黄石市创新发展中心党组书记、主任赵延忠带队赴湖北理工学院同学院党委副书记、院长胡国珍座谈，围绕创新人才引育、科技成果转化、科创项目共享等内容进行详细交流。
```

示例2（日期范围+多地点）：
```
为积极对接粤港澳大湾区优质产业创新资源，10月27日至30日，市政府党组成员闫树同志一行赴深圳开展招商考察，先后拜访强脑科技（深圳分公司）、东方富海、同创伟业等创投机构和未来产业科技公司。
```

示例3（时间+地点+活动）：
```
3月18日下午，2026年"武创源"颠覆性技术创新大赛黄石专场推介会在黄石市智能显示产业园成功举办。
```

**导语长度**：控制在80-150字。

#### 第三段：主体

**内容组织**：

##### 1. 被访方致欢迎辞

```
[地点/单位]对市创新发展中心一行的到访表示欢迎。他表示，[对中心的评价]，[合作期望]。

示例：
东楚投资集团党委副书记、总经理徐海涛对市创新发展中心一行的到访表示欢迎。他表示，市创新发展中心作为推动全市创新发展、产业升级的重要力量，在创新资源整合、项目招引培育等方面有着丰富经验，此次专题交流搭建了良好的对接平台，期待与中心深度联动......
```

##### 2. 被访方介绍情况

```
[领导姓名]详细/首先介绍了[单位/园区/企业]在[领域]的[发展情况/核心技术/建设成果]。[必要时补充背景]。

示例：
东楚投资集团低空公司负责人详细介绍了公司发展情况及全市低空应用场景落地推动进展......
```

##### 3. 我方分享经验/介绍工作

```
[我方负责人]首先分享了[前期工作/考察成果]，[具体内容]。[结合实际]重点介绍了[工作思路/计划/大赛设置]。

示例：
市创新发展中心科技招商科负责人首先分享了前期赴苏州考察低空经济的先进经验，苏州在无传统民航机场的基础上，以"制度先行、平台筑基、场景落地、产教融合"为核心，探索出全国领先低空经济生态的实践路径。并结合考察成果与黄石产业实际，重点介绍了2026年黄石未来产业创新创业大赛办赛思路......
```

##### 4. 领导讲话引用格式

```
[领导姓名]指出/[表示]/[强调]/[认为]，[具体内容]。[必要时补充意义]。

示例：
赵延忠指出，随着新能源汽车产业的快速发展，废旧动力电池的回收利用具有重大的环保和经济价值。
```

##### 5. 补充企业/园区介绍（招商考察类）

```
[企业名称]是一家/专注于/掌握......的企业/公司，在[领域]有着独特优势/取得XX成绩......

示例：
旻太航空科技是武汉首家载人eVTOL研发企业，负责人明确计划将研发中心留在武汉、生产基地优先选址黄石。
```

#### 第四段：具体工作方向

**结构**：序号词 + 领域 + 措施

**序号词使用规则**：
1. 首选自然流畅："一是、二是、三是......" 或 "一要、二要、三要......"
2. 次选："第一、第二、第三......"
3. 领导讲话常用："一要、二要、三要......"（带部署性质）
4. 避免：大写"一、二、三"（过于正式公文感）

**方向分类与常用句式**：

| 方向 | 常用句式 |
|------|----------|
| 产业集聚 | 共同推动XXX产业落地/布局，完善全链条产业生态 |
| 创新项目 | 共同招引XXX领域先进技术/优质市场主体，补齐产业链短板 |
| 赛事举办 | 联合办好XXX大赛......以赛引才、以赛兴业 |
| 基金/资本 | 共同培育XXX领域优质初创项目，以资本赋能产业高质量发展 |
| 合作共赢 | 共同完善XXX制度规则体系、拓展多元化应用场景 |
| 平台建设 | 共建创新平台、推动成果就地转化 |
| 人才引育 | 培养和引进高端人才 |
| 服务保障 | 全流程、一站式服务/全要素保障 |

**示例**：

赵延忠表示，下一步双方深化合作、共赢发展：
```
一是聚焦产业集聚，共同推动低空智造产业落地、低空教培业务布局，完善全链条产业生态；
二是聚焦创新项目，共同招引低空经济领域先进技术和优质市场主体，补齐产业链短板；
三是聚焦赛事举办，联合办好黄石未来产业创新创业大赛低空经济专项赛道，充分发挥赛事桥梁纽带作用；
四是聚焦基金设立，共同培育低空经济领域优质初创项目；
五是聚焦合作共赢，共同完善低空经济发展制度规则体系、拓展多元化应用场景。
```

#### 第五段：结语

**标准套路**：

```
此次[活动名称]务实高效，双方围绕[重点内容]深入交流，进一步明晰了[合作方向]、凝聚了[发展共识]。双方一致表示，将[长期规划/建立机制]，[首批具体行动]，[最终目标]。
```

**结语要素**（四件套）：
1. 肯定活动成效：务实高效、深入交流、精准对接
2. 点明交流成果：明晰方向、凝聚共识、达成共识
3. 表述后续行动：建立机制、启动对接、梳理清单
4. 展望最终目标：助力......提升/发展、注入创新动能

**示例**：

```
此次对接交流务实高效，双方围绕未来产业赛事联动、科创平台共建、科技招商协同等重点内容深入交流，进一步明晰了合作方向、凝聚了发展共识。双方一致表示，将建立长期深度战略协作关系，启动日常对接机制，梳理首批合作项目清单，以协同创新激活发展动能，助力区域创新生态持续优化、产业能级稳步提升。
```

## 四、语言组织规范

### 常用动词

| 场景 | 常用动词 |
|------|----------|
| 访问/考察 | 赴、走访、实地调研、深入走访 |
| 座谈交流 | 围绕、展开、进行、共商 |
| 听取介绍 | 听取、详细了解、全面了解 |
| 表示态度 | 指出、强调、表示、认为 |
| 推动合作 | 深化、共建、携手、联动 |
| 取得成效 | 走在前列、实现突破、取得进展 |
| 取得成果 | 达成合作意向、顺利落地、签约注册 |
| 发出邀请 | 诚挚邀请、诚邀、热忱欢迎 |

### 常用修饰词

| 类型 | 修饰词示例 |
|------|------------|
| 形容工作力度 | 深度践行、全力打造、持续深化、扎实推进 |
| 形容发展态势 | 势头迅猛、稳步提升、再上新台阶、走在前列 |
| 形容合作基础 | 具有坚实基础、开放共赢、优势互补 |
| 形容工作成效 | 务实高效、精准对接、深入交流、成果丰硕 |
| 形容活动规格 | 高端、高效、务实、热烈 |

### 常用固定搭配

```
为......，深入推进/加快推进
围绕......，展开深入座谈/凝聚发展共识
双方围绕......深入交流
进一步明晰......合作方向
凝聚......发展共识
以......为核心/抓手
推动......落地见效/落地生根
以赛促引、以赛助创、以赛兴产
以赛事为纽带
打通......"最后一公里"
构建......创新生态/发展格局
锚定......定位/目标
聚焦......领域/方向
发力......
```

### 过渡词使用

| 类型 | 过渡词 |
|------|--------|
| 段内过渡 | 随后、在此基础上、与此同时 |
| 观点过渡 | 他表示、赵延忠指出、会上 |
| 总结过渡 | 总体来看、基于上述讨论 |

## 五、人物称呼规范

### 首次出现

```
[职务]+[姓名]（首次全称）
```

示例：
- "市创新发展中心党组书记、主任赵延忠"
- "东楚投资集团党委副书记、总经理徐海涛"
- "市政府党组成员闫树同志"

### 后续称呼

- **正式场合**：仅用职务（如"赵延忠表示"）
- **一般场合**：仅用姓名（如"郝义国围绕......进行了全面介绍"）

### "一行"的使用

指代我方/客方考察团组：
```
赵延忠一行实地调研XXX公司
赵延忠一行参观园区综合楼展厅
```

### 多人出现时的处理

先介绍主要人物，再提及随行人员：

```
市创新发展中心党组书记、主任赵延忠，[随行人员职务+姓名]参加座谈。
市创新发展中心副主任秦世昆，市招商服务中心、市国投集团相关负责人参加此次考察。
```

## 六、不同类型简讯的写作要点

### 1. 招商考察类

**侧重点**：
- 介绍被考察企业的核心技术/产品
- 表达合作意向和诉求
- 说明我方产业基础和优势

**特殊句式**：
```
[企业]掌握了......核心技术/颠覆性技术
致力于突破......"卡脖子"技术瓶颈
推动......国产化替代
诚邀/诚挚邀请[企业]来黄布局/落户
```

**结构特点**：
1. 导语：时间+人物+地点+目的
2. 主体：走访企业→体验/参观→座谈交流→合作探讨
3. 结语：肯定成效+后续对接

### 2. 座谈交流类

**侧重点**：
- 双方领导讲话要点
- 达成的共识
- 明确的下一步合作方向

**特殊句式**：
```
双方围绕......进行深入座谈
就......深入探讨
达成初步/合作共识
```

**结构特点**：
1. 导语：时间+人物+地点+活动+目的
2. 主体：被访方致欢迎辞+介绍情况→我方分享经验+介绍工作→领导讲话
3. 结语：务实高效+明晰方向+凝聚共识

### 3. 调研活动类

**侧重点**：
- 实地走访的企业/园区情况
- 调研中发现的问题
- 协调解决的措施

**特殊句式**：
```
先后深入/实地走访XXX、XXX等重点企业
以实地调研摸清产业发展实情
以精准服务破解企业发展难题
```

**结构特点**：
1. 导语：时间+人物+地点+活动+目的
2. 主体：走访企业A（介绍+点评）→走访企业B→走访企业C
3. 结语：肯定+服务承诺

### 4. 赛事活动类

**侧重点**：
- 大赛背景和目的
- 赛事设置和赛道
- 以赛促引的工作思路
- 已有成果展示

**特殊句式**：
```
大赛专设XXX专项赛道
聚焦XXX、XXX等细分领域
强化与XXX合作
依托XXX优势
通过"以赛促引、以赛助创、以赛兴产"模式
吸引......优质项目、先进技术、高端人才集聚
推动赛事流量转化为发展增量
```

**结构特点**：
1. 主标题（往往是亮点概括）
2. 导语：活动背景+时间地点
3. 主体：主办方致辞→推介内容→领导讲话
4. 合作方向（序号式）
5. 结语：展望+目标

### 5. 协同创新类

**侧重点**：
- 双方资源优势
- 合作共建内容
- 未来发展愿景

**特殊句式**：
```
深化......协同创新
加快......项目集聚落地
建立常态化对接机制
携手推动......高质量发展
```

### 6. 工作推进/筹备会类

**侧重点**：
- 听取汇报的内容
- 提出的工作要求
- 明确责任分工

**特殊句式**：
```
会议听取/通报了......情况
就......提出明确要求
下一步，将....../各单位将......
```

**结构特点**：
1. 标题（往往直接点明会议名称）
2. 导语：时间+主持+参加
3. 主体：汇报内容（序号式）→领导要求（序号式）
4. 结语：工作部署

## 七、常见问题与修正

### 问题1：导语过长

**问题**：导语堆砌过多信息，显得冗长。

**修正**：导语控制在100字以内，只保留核心信息。

### 问题2：领导讲话堆砌

**问题**：连续多段都是领导讲话，缺乏衔接。

**修正**：
- 精简每位领导的讲话（保留1-2个核心观点）
- 用"随后"、"接着"等过渡
- 适当加入活动描写（参观、座谈等）

### 问题3：序号过多

**问题**：合作方向列了6、7条以上，重点不突出。

**修正**：
- 精选3-5条最核心的方向
- 其他内容合并到主体叙述中
- 或分层次：主要方向（前三条）+ 次要方向（简略带过）

### 问题4：结语空洞

**问题**：结语喊口号，没有实质性内容。

**修正**：
- 结语要有具体的后续行动
- 明确"首批合作项目清单"等具体事项
- 避免"将不断深化合作"等空泛表述

### 问题5：标题平淡

**问题**：标题没有亮点，不能吸引读者。

**修正**：
- 优先使用对仗式主标题
- 副标题要清晰完整
- 可以适当使用符号（如"│"）增加层次感

## 八、写作检查清单

写完简讯后，逐项检查：

- [ ] 标题主副搭配，主标题有亮点/对仗感
- [ ] 导语包含时间、人物、地点、活动、目的
- [ ] 导语长度80-150字
- [ ] 领导讲话精炼，保留核心观点（每条1-2句）
- [ ] 序号方向3-5条为宜

## 十、高级写作技巧

### 1. 让主标题更有文采

**技巧**：使用动词对仗，形成动宾结构

- "深化**协同创新**，共筑**发展高地**"
- "深化**赛事成果**，激发**创新活力**"
- "以**赛引赛**融资源，**赛事协同**筑高地"

### 2. 导语信息排序

标准顺序：**目的 → 时间 → 人物 → 地点 → 活动**

### 3. 数据的巧妙运用

在简讯中适时加入数据，增强说服力：
- "推动7个获奖项目与黄石达成合作"
- "累计对接项目超600个"
- "已完成全市35个无人机起降点位布设"

### 4. 借用主办方/企业的话表达观点

间接引用比直接陈述更有说服力：
```
"黄石不仅有扎实的装备制造基础和完备的产业链体系，更有全链条的产业支持政策，与我们企业的产业化需求高度契合。"
——[企业名称]负责人
```

### 5. 善用"四字格"

政府公文常用的四字格表达：
- 精准对接、深入交流
- 务实高效、共谋发展
- 协同联动、聚力前行
- 创新驱动、提质增效
